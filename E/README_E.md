# 目標E：運動発達（自発運動 → 目的的運動 → 歩行）

太郎の**運動（粗大運動）の発達**を、人間の乳児の順序で育てる段階。運動性喃語（自発運動）から始まり、その**終了（目的的運動への移行）**を経て、座位・這い這い・歩行までを扱う。

## なぜ独立の目標か（＝依存関係）
[発達依存関係マップ](../doc/発達依存関係マップ.md)により、運動発達は**C（自己モデル）とD（他者理解）の"土台"**と判明：
- 自己モデル(C)は運動性喃語が育てる（Hadders-Algra 一次変動性）
- D（動く自分での視覚的他者理解＝egomotion）は、**運動性喃語の終了＋自己運動の割り引きが前提**
- ＝運動発達を飛ばしてDをやったのがDの壁の原因。だからDの前提として運動発達を独立目標に切り出した。

> ⚠️ 文字「E」は**識別子**であり発達順ではない。依存関係では**運動発達はC・Dより"前"の土台**。

## スコープ（[運動発達ロードマップ](../doc/運動発達ロードマップ.md)の8段階）
運動性喃語(0-2m) → 頭部制御 → リーチ → 寝返り → 座位 → 這い這い → つかまり立ち → 歩行(12-18m)。
**最初のマイルストーン＝「運動性喃語の終了（目的的運動への移行）」**（一次変動性→二次変動性）。

## 哲学（太郎と整合）
各段階を**スクリプトせず**、**体年齢（成長モジュール）＋態勢＋環境（足場）**を与えて、同じ本能（予測誤差最小化・好奇心・強化学習・睡眠リプレイ）で運動が創発するかを見る。判定は**動画目視＋物理指標（jerk等）**。

## C・Dとの関係
- **C（自己内部モデル）**：運動発達が育てる。C5（運動性喃語の質・progress報酬でmean\|jerk\| 65%減）はここに接続。
- **D（他者理解）**：運動発達＋egomotion割り引きが済むまで**"待ち"**。
