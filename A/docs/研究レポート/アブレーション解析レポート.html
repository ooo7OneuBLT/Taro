1. 実験概要

手法：アブレーション（損傷実験）

神経科学では、脳の一部を損傷させて「何ができなくなるか」を観察することで、その部品の機能を明らかにする。太郎でも同じ手法を適用し、6つの部品を1つずつ無効化して、ベースライン（全部品あり）との差を測定した。

条件

  • full — 全部品あり（ベースライン）
  • no_brocas — ブローカ野なし：発話計画をせず1文字ずつ生成（≒ブローカ失語）
  • no_cerebellum — 小脳なし：運動記憶（順モデル・逆モデル）なし（≒運動失調）
  • no_lc — 青斑核なし：NE固定（0.5）、探索/搾取の自動調整なし（≒覚醒障害）
  • no_habit — 馴化なし：飽きのペナルティなし（≒馴化障害）
  • no_social — 社会的報酬なし：親が笑わない（≒ネグレクト環境）
  • no_coupling — 連動制御なし：最初から全パラメータ独立（≒早熟な発達）

共通設定

  • 訓練フレーズ：まま, ばば, ないない, まんま, ねんね
  • 転移テストフレーズ：ぱぱ, だだ, いやいや, ぶぶ, にに
  • 最大ターン数：2,000
  • Stage 4達成条件：模倣スコア≥0.8が10回連続

6. 考察

致命的な部品（なくすとStage 4に到達できない）

ブローカ野（発話計画）

模倣スコア 0.939→0.264（-72%）。完全一致率は0%に壊滅。発話計画なしの1文字ずつ生成では、フレーズ長の制御も音の順序制御もできない。人間のブローカ失語（理解はできるが流暢に話せない）と一致する症状。太郎の最重要部品。

青斑核（NE探索制御）

模倣スコア 0.939→0.708（-25%）。完全一致率は58%→7%。NE固定（0.5）では報酬が低下しても探索レベルが上がらず、固着から回復できない。人間の覚醒障害・注意障害と類似のパターン。学習の安定性に必須。

冗長な部品（なくても現時点では影響が小さい）

小脳（運動記憶）

むしろ微妙に性能向上（完全一致58→67%）。発達途中の逆モデルが不正確な運動指令を出し、かえって足を引っ張っている可能性がある。ただしフレーズが複雑化する将来のStageでは重要になる可能性が高い。

馴化（飽き）

ほぼ同等の性能。青斑核が「報酬が低いと探索を増やす」機能で、馴化の「同じ出力に飽きる」機能を代替していると考えられる。機能的冗長性（ロバストネス）を提供している。

社会的報酬（親の笑顔）

ほぼ同等。模倣報酬（w=0.5）と予測報酬（w=0.3）だけで十分学習できている。社会的報酬の重み（w=0.2）が小さいことが一因。ただしこれは「ネグレクトでも学べる」ことを意味するわけではなく、現在のタスク（オウム返し）が単純すぎる可能性がある。

連動制御（Frame-Content）

ほぼ同等。2,000ターンの時点では差が出にくい。連動制御の主な効果は初期の探索空間削減であり、Stage 0-1の学習速度に影響するが、長期的な到達点には影響しない可能性がある。

全体的な知見

  • 「計画して話す」が最も重要 — 1文字ずつの逐次生成ではオウム返しは不可能。フレーズ全体を先に計画し、バッファから順次実行する仕組み（ブローカ野/GODIVAモデル）が決定的。
  • 「困ったら探索を増やす」が2番目に重要 — 固定の探索率では局所解に陥る。報酬の変動に応じて探索/搾取を動的に切り替える仕組み（青斑核/NE系）が安定学習に不可欠。
  • 冗長な部品にも意義がある — 馴化と青斑核は機能的に重複しており、一方が壊れてももう一方が補償する。この冗長性は人間の脳の設計原則と一致する。
  • タスクの複雑さに依存 — 現在のタスク（2〜4モーラのオウム返し）は単純であり、小脳・社会的報酬・連動制御の貢献が見えにくい。より長いフレーズ、より多様な音韻構造、意味理解を求めるタスクでは結果が変わる可能性がある。

7. 実験手法

各条件は独立したTaroEnvironmentインスタンスで実行。同じ乱数シード制御は行っていない（確率的な変動あり）。

  • ベースライン：A2-11の全部品有効構成
  • 無効化方法：TaroEnvironment(ablation={"部品名": True})
  • 各条件で同一のフレーズセット・同一のターン数を使用
  • 転移テスト：訓練後に未見フレーズを各5回試行
  • 実行環境：RTX 3060 Ti, Python 3.12, PyTorch 2.11

実験スクリプト：src/environment/ablation.py

詳細データ：logs/ablation_results.json

8. 次のステップ

  • 複数回試行の平均 — 乱数による変動を排除するため、各条件を3〜5回繰り返して平均・標準偏差を取る
  • より長いターン数 — 5,000〜10,000ターンで小脳や社会的報酬の長期的効果を検証
  • 複合アブレーション — 2つ以上の部品を同時に無効化して相互作用を調べる（例：馴化＋青斑核を同時に除去）
  • 段階的な難易度 — より長いフレーズ、より多様な音韻で小脳・連動制御の貢献を引き出す
