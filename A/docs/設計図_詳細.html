太郎プロジェクト 詳細設計図

各.pyファイルの中身・変数の流れ・関数の呼び出し関係 / 2026-06-27

1. ファイル構成と各ファイルの役割

環境（外の世界）— environment/

core.py — TaroEnvironmentクラス司令塔

全部品をまとめて「1ターンのループ」を提供する。
持っているもの：brain(大脳皮質), cerebellum(小脳), brocas_area(ブローカ野), learner(大脳基底核), dopamine, habituation, locus_coeruleus, vocal_tract(声道), stamina(体力), vocab(語彙)
メイン関数：step(parent_text, r_social) → 1ターン実行して結果を返す

parent_sim.py — 親シミュレータ

ParentSmile: compute_smile(r_imit, taro_output) → r_social(笑顔の度合い 0〜1)
StageTracker: check_all(turn) → 発達段階の達成判定
run_simulation(): 全体の実験ループ（何千ターンも回す）

ablation.py — アブレーション解析

各部品を1つずつ無効化して比較実験を行う。
TaroEnvironment(ablation={"部品名": True}) で部品を無効化

太郎の脳 — taro/brain/

cortex.py — TaroBrainクラス（大脳皮質）中核

聞く：forward_hidden(x, hidden) → 入力を処理して隠れ状態hを更新
予測する：forward_perception(x, hidden) → 次の文字の確率logitsを出力
口の指令：forward_articulation(h) → 4つのヘッドから口パラメータ(pl, ml, vl, vol)を出力
話す：generate(hidden, ..., speech_plan, ne_level, cerebellum)
  → 発話計画に沿って文字列を生成。戻り値は(generated, log_probs, hidden)
内部処理：
  _choose_param(logits, allowed) — 脳の意図（ノイズなし）
  _apply_ne_noise(value, ne_level, allowed) — NEで1つ隣にずらす
  _log_prob_of(logits, value, allowed) — 小脳指定の値のlog確率

left_frontal_lobe/brocas_area.py
BrocasArea（ブローカ野）必須

plan(parent_text, cerebellum, vocal_tract)
→ 親の発話を1文字ずつ分解
→ 小脳にlookup_motor(char)で口の動きを問合せ
→ motor_bufferにリスト格納
has_next() / next_motor()
→ バッファから1つずつ取り出す
バッファが空 → 発話終了

cerebellum.py
Cerebellum（小脳）

learn_from_experience(place, manner, voicing, vowel, char)
→ forward_map[(p,m,v,vw)] = char
→ inverse_map[char] = (p,m,v,vw)
lookup_motor(char) → 逆引き
predict_sound(p,m,v,vw) → 順引き
experience_countで確信度を蓄積

basal_ganglia.py
TaroLearner（大脳基底核）

learn_perception(input_tokens)
→ 次トークン予測の誤差p_lossと予測確率pred_probsを返す
learn_action(log_probs, delta)
→ a_loss = -delta * Σlog_prob（REINFORCE）
update(p_loss, a_loss)
→ 合算してbackpropで重み更新 ⚠️逸脱

hippocampus.py
Hippocampus（海馬）未実装

将来の睡眠リプレイ・記憶定着用。
現在は空のクラスのみ。

本能 — taro/brain/instincts/

imitation.py — 模倣衝動

compute_imitation_reward(parent_tokens, taro_tokens, vocab, vocal_tract)
→ 重み付き編集距離で類似度を計算
→ r_imit (0〜1) を返す
声道のparam_distance()で「口が近い音」のコスト小

prediction.py — 予測

compute_prediction_reward(pred_probs, actual_tokens)
→ 各トークンの予測確率の平均
→ r_pred (0〜1) を返す

dopamine.py — ドーパミン

compute_rpe(reward)
→ delta = reward - baseline
→ baselineを更新（移動平均 0.95）
→ delta（報酬予測誤差）を返す

reward.py — 報酬合算

compute_total_reward(r_imit, r_pred, r_social, r_habit, weights)
→ R = 0.5*r_imit + 0.3*r_pred + 0.2*r_social + r_habit
→ max(0, R) を返す

habituation.py — 馴化（飽き）

compute_penalty(taro_text)
→ 同じ出力の繰り返し回数を見る
→ r_habit (0〜-1) を返す

locus_coeruleus.py — 青斑核必須

observe_reward(R) → 報酬を記録（直近20ターン）
release_ne() → NEレベルを調整
  報酬低い(<0.1) → NE↑ / 高い(>0.3) → NE↓
get_ne_level() → ne_level (0.1〜1.0)

太郎の身体 — taro/body/

vocal_tract.py — VocalTract（声道）

speak(place, manner, voicing, vowel)
→ 4パラメータから文字に変換（例：1,1,1,0 →「ま」）
hear(char) → 文字からパラメータに逆変換
get_allowed() → 現在のStageで使える値
is_coupled() → 連動モード？
update_stage(sim_seconds, ...) → 時間で成熟
param_distance(a, b) → 2文字の口の距離(0〜4)

stamina.py — Stamina（体力）

get() → 現在の体力値（最大何文字出せるか）
grow() → 毎ターン+0.001ずつ成長
初期値3 → 最大15

補助ツール — src/

sim_clock.py — シミュレーション内時計。1ターン=5秒

archive.py — 脳の状態をファイルに保存/読込

logger.py — 毎ターンのデータをJSONLに記録＋グラフ描画

config/config.yaml — 設定値（脳の大きさ、報酬の重みなど）

2. 1ターンの詳細フロー（変数の流れ）

以下は TaroEnvironment.step("まま", r_social=0.5) が呼ばれたときの処理を順に追ったもの。

1
入力を受け取る
parent_text = "まま" を受け取る
vocab.encode("まま") → parent_tokens = [5, 5]（「ま」のID=5だとすると）
BOS(=1)とEOS(=2)をつけて full_tokens = [1, 5, 5, 2]
▼
2
知覚学習（大脳基底核 → 大脳皮質）
learner.learn_perception(full_tokens) が呼ばれる
内部：brain.forward_perception([1, 5, 5]) で次のトークンを予測
正解 [5, 5, 2] と比較して予測誤差 p_loss を計算
各位置での予測確率 pred_probs = [tensor(...), tensor(...), ...] も返す
出力される変数：
p_loss = 予測誤差（数値）
pred_probs = 各位置の予測確率のリスト
▼
3
聞く（大脳皮質）
brain.forward_hidden(listen_input)
「まま」全体を聞いて、脳の状態 h（隠れ状態）を更新する
この h が「何を聞いたか」の短期記憶になる
出力される変数：
h = 隠れ状態（128次元のベクトル = 脳の状態）
▼
4
発話計画を立てる（ブローカ野 ← 小脳）
brocas_area.plan("まま", cerebellum, vocal_tract)
1文字ずつ処理：
  「ま」→ cerebellum.lookup_motor("ま") → (1,1,1,0)（唇・鼻音・有声・あ）
  「ま」→ cerebellum.lookup_motor("ま") → (1,1,1,0)
結果：motor_buffer = [{motor:(1,1,1,0), known:True}, {motor:(1,1,1,0), known:True}]
structure = 2（2モーラ分の計画）
出力される変数：
motor_buffer = 口の動きのリスト（計画）
structure = 計画の長さ（＝親の発話のモーラ数）
▼
5
発声する（大脳皮質 → 声道）
brain.generate(hidden=h, ..., speech_plan=brocas_area, ne_level=0.5, cerebellum=cerebellum)

ループ（2回 = 計画が2モーラだから）：
  ① brocas_area.next_motor() → {motor:(1,1,1,0)}
  ② 脳の意図を計算：forward_articulation(h) → ヘッドから確率分布
  ③ 小脳の指示の log 確率を記録：_log_prob_of(pl, 1, allowed_place) → log_prob
  ④ NEノイズ注入：_apply_ne_noise(1, 0.5, [0,1])
    → 確率35%（0.5×0.7）で隣にずれる → 例えば s_place=1のまま or s_place=0に
  ⑤ 声道で発声：vocal_tract.speak(1, 1, 1, 0) → "ま"
  ⑥ 小脳に記録：cerebellum.learn_from_experience(1, 1, 1, 0, "ま")
  ⑦ 自分の声を聞く：forward_hidden(token("ま"), h) → hを更新
  → 2回目も同様

バッファが空 → ループ終了
出力される変数：
generated = [5, 5]（「ま」「ま」のトークンID）
log_probs = [tensor(-0.3), tensor(-0.2)]（各文字の選択確率のlog）
taro_text = "まま"（vocabでデコード）
▼
6
報酬を計算する（本能たち）
4つの報酬を並列に計算：
① compute_imitation_reward(parent_tokens=[5,5], generated=[5,5], vocab, vocal_tract)
  → 完全一致なので r_imit = 1.0
② compute_prediction_reward(pred_probs, parent_tokens)
  → 予測がどれだけ当たったか r_pred = 0.6（例）
③ r_social = 0.5（親から受け取った笑顔の度合い）
④ habituation.compute_penalty("まま")
  → 初回なので r_habit = 0.0（飽きなし）

合算：
compute_total_reward(1.0, 0.6, 0.5, 0.0, weights)
→ R = 0.5×1.0 + 0.3×0.6 + 0.2×0.5 + 0.0 = 0.78
出力される変数：
r_imit = 1.0, r_pred = 0.6, r_social = 0.5, r_habit = 0.0
R = 0.78（合計報酬）
▼
7
学習する（ドーパミン → 大脳基底核）
ドーパミン（報酬予測誤差）：
dopamine.compute_rpe(R=0.78)
→ delta = 0.78 - baseline（例：baseline=0.5 → delta = 0.28）
→ 「思ったより良かった！」= 正のδ
→ baselineを更新：0.95×0.5 + 0.05×0.78 = 0.514

行動学習（REINFORCE）：
learner.learn_action(log_probs, delta=0.28)
→ a_loss = -(0.28) × (log_prob₁ + log_prob₂)
→ delta > 0 なので「この発話を出しやすくする」方向に重みを調整

重み更新：
learner.update(p_loss, a_loss)
→ total_loss = p_loss + a_loss → backpropで脳の重みを更新
出力される変数：
delta = 0.28（ドーパミン = 報酬予測誤差）
pl, al（知覚損失、行動損失の数値）
▼
8
次のターンの準備
声道の成熟：
vocal_tract.update_stage(sim_seconds, 300, 900, 1500, 1200)
→ 5分経過でStage 1, 15分でStage 2, ...（時間で自動進行）

青斑核のNE調整：
locus_coeruleus.observe_reward(R=0.78) → 報酬を記録
locus_coeruleus.release_ne()
→ 直近20ターンの平均が0.3以上なので NE↓（探索を減らす）
brain.receive_ne(ne_level) → 次のターンで使うNEを脳に渡す

体力成長： stamina.grow() → +0.001
時計： clock.tick() → +5秒
🔄 次のターンへ（Step 1に戻る）

3. 変数の受け渡しまとめ

parent_text → vocab → parent_tokens → cortex → h(隠れ状態)
parent_text → brocas_area.plan() ← cerebellum → motor_buffer(発話計画)
h + motor_buffer + ne_level → cortex.generate() → vocal_tract.speak() → taro_text
parent_tokens + generated → imitation → r_imit
pred_probs + parent_tokens → prediction → r_pred
taro_text → habituation → r_habit
r_imit + r_pred + r_social + r_habit → reward → R
R → dopamine → delta
delta + log_probs → basal_ganglia → 脳の重みを更新
R → locus_coeruleus → ne_level（次ターンで使う）
