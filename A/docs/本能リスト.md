# 太郎の本能リスト

太郎に与えた本能の一覧。各本能について、人間での根拠・実装場所・有効な発達段階を記録する。

---

## A1で有効な本能

### 1. 模倣衝動（真似したい）

| 項目 | 内容 |
|------|------|
| **人間での根拠** | 新生児模倣（Meltzoff & Moore, 1977）。ただし近年反論あり（Oostenbroek et al., 2016）。学術的に未決着のため、本研究では仮説的な本能として採用 |
| **場所** | 太郎の中身（本能） |
| **実装** | 親の発話と太郎の出力の類似度 → 内的報酬 r_imit ∈ [0, 1] |
| **有効段階** | A1〜（報酬量は固定だが、他の報酬源が育つと相対的に小さくなる＝創発的減衰） |
| **ラベル** | 【人間模倣】 |

### 2. 予測（予測誤差最小化）

| 項目 | 内容 |
|------|------|
| **人間での根拠** | 予測処理 / 自由エネルギー原理（Friston）。脳は次の入力を絶えず予測し、予測が当たると安定する |
| **場所** | 太郎の中身（本能） |
| **実装** | 次トークンの予測が的中するほど報酬 r_pred |
| **有効段階** | A1〜（恒久） |
| **ラベル** | 【人間模倣】 |

### 3. 報酬予測誤差 = ドーパミン

| 項目 | 内容 |
|------|------|
| **人間での根拠** | ドーパミンニューロンが「もらえた報酬 − 予想した報酬」を発火する（Schultz, 1997）。強化学習のTD誤差と数式上一致 |
| **場所** | 太郎の中身（本能） |
| **実装** | 学習信号 δ = 実報酬R − baseline（報酬予測）。全学習を駆動する |
| **有効段階** | A1〜（恒久） |
| **ラベル** | 【人間模倣】＝【既存AI研究】 |

### 4. 社会的報酬（親の反応で喜ぶ）

| 項目 | 内容 |
|------|------|
| **人間での根拠** | 新生児は笑顔・親の声・対乳児発話・視線・随伴性に生まれつき反応する |
| **場所** | 環境（親の入力） |
| **実装** | 親の「笑顔の強さ」を連続値 r_social ∈ [0, 1] で入力 |
| **有効段階** | A1〜（恒久。後に注目・呼応等のバリエーション追加可） |
| **ラベル** | 【人間模倣】 |

## 発達で消える本能（将来の検討事項）

人間の乳児には「把握反射」「モロー反射」等、生後4〜6か月で消失する原始反射がある。テキストの太郎に直接対応するものは少ないが、「足場として働き役目を終えると弱まる本能」という原理は、模倣衝動の相対的減衰として自然に再現される。
