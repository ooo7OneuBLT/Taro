# A1 設計メモ

A1（オウム返し試作）の技術仕様と、各判断の人間模倣根拠を記録する。

---

## 太郎の脳

**予測する単一の再帰ネットワーク**

- 会話ストリームの**次の文字（トークン）を予測**し続ける1つのネット
- 「翻訳装置（seq2seq）」のような人間にない構造は持たない【人間模倣】
- 構成：GRU（埋め込み64、隠れ128）。人間の脳（860億ニューロン）より桁違いに小さい（パラメータ約10万＝シナプス比で約10億分の1）
- オウム返しは内蔵モジュールでなく、本能（報酬）に導かれて**行動として創発**する

### 出力のゆらぎ（喃語）

- 出力に温度τを持たせ、確率的にサンプリング
- 初期はτ高い＝赤ちゃんの喃語（運動探索）【人間模倣】
- 発達が進むとτ減衰＝発音安定。これ自体が発達の再現

## 報酬の合成

```
R = w_imit × r_imit + w_pred × r_pred + w_social × r_social
```

| 記号 | 意味 | 範囲 | 算出 |
|------|------|------|------|
| r_imit | 模倣報酬（親の発話との類似度） | [0, 1] | 自動（本能） |
| r_pred | 予測報酬（次トークン予測の的中） | [0, 1] | 自動（本能） |
| r_social | 社会的報酬（親の笑顔の強さ） | [0, 1] | 手動（親の入力） |
| w_* | 各報酬の重み | 設定可能 | config.yaml |

### 学習信号（ドーパミン）

```
δ = R − baseline
```

- baseline = 報酬の移動平均（＝「予想していた報酬」＝報酬予測）
- δ > 0 → 予想より良かった → この行動を増やす
- δ < 0 → 予想より悪かった → この行動を減らす
- これがドーパミンの報酬予測誤差（Schultz, 1997）と同じ仕組み【人間模倣】＝【既存AI研究】

---

## 学習メカニズム

### 知覚の学習

次トークン予測の誤差（cross-entropy loss）を最小化＝予測処理【人間模倣】

### 行動の学習

方策勾配（REINFORCE）：高報酬だった発話を出やすくする更新

```
loss = −(R − baseline) × log_prob(生成した系列)
```

### ⚠️ 逸脱の明記

重み更新に**誤差逆伝播（backprop）**を使用する。これは脳が厳密には行っていないとされる手法【既存AI研究・⚠️逸脱】。「報酬で重みを調整する」大枠は脳と同じ。生物的学習則（ヘブ則等）への置換はA2以降の課題。

## 「太郎の中身」と「環境」の仕分け

| 要素 | 場所 | 根拠 |
|------|------|------|
| 4本能（模倣・予測・ドーパミン・社会的報酬） | 中身 | 実在の本能【人間模倣】 |
| 短くやさしい発話（カリキュラム） | 環境：親の話し方 | 対乳児発話 motherese【人間模倣】 |
| 限られた語彙 | 環境 | 親が乳児に使う語彙は限られる【人間模倣】 |
| 発話が長くなる | 環境主導で創発 | scaffolding【人間模倣】 |

## シミュレーション内時間

- **イベント駆動**：1回のやり取り＝sim時間で数秒を加算（設定可能）
- 発達年齢＝やり取り時間の合計
- 計算速度に依存せず再現性が高い
- ⚠️ 実際の乳児はやり取りの合間も音を浴びている。明示的やり取りだけ数えるとsim年齢はゆっくり積み上がる

## 用語集

| 用語 | 説明 |
|------|------|
| **強化学習（RL）** | ごほうびが増える行動を試行錯誤で学ぶ仕組み |
| **方策勾配（REINFORCE）** | 「この行動でごほうびが多かった → この行動の確率を上げる」更新方法 |
| **報酬予測誤差（RPE / δ）** | もらえた報酬 − 予想した報酬。ドーパミンの発火パターンと一致 |
| **baseline** | 「普段どれくらい報酬がもらえるか」の予想値。移動平均で算出 |
| **温度（τ）** | 出力のばらつき度合い。高い＝喃語（ランダム）、低い＝安定した発話 |
| **cross-entropy loss** | 予測した確率分布と実際の結果のズレを測る尺度 |
| **誤差逆伝播（backprop）** | ネットの出力の誤差を入力側に遡って伝え、重みを調整する方法 |
| **GRU** | 時系列データを扱う再帰ネットワークの一種。過去の情報を記憶しながら処理する |
