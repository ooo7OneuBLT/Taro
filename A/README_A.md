# 目標A：オウム返し

親（研究者）の発言を太郎が同じように返す。人間でいう「生後まもなく〜数か月の、親の発声を模倣する段階」。

## 検証仮説

> 人間の乳児が持つとされる4つの本能のみを与えた場合、あらかじめ答えを教え込まなくとも、太郎は親の発話を模倣する行動（オウム返し）を自ら獲得する。

## 成功基準

親の入力と完全一致する出力を**連続10回**達成。

## 結果

✅ **達成**。Stage 4到達（約1050ターン）。訓練フレーズ完全一致率45〜50%、
**未見フレーズへの転移テストで80〜100%完全一致**。
アブレーション解析（A2-12）：ブローカ野・青斑核（NE系）が必須、他4部品
（小脳・馴化・社会的報酬・連動制御）は現タスクでは冗長と判明
（人間には存在する機能なので削除はしない方針）。
