# 目標A：オウム返し

親（研究者）の発言を太郎が同じように返す。人間でいう「生後まもなく〜数か月の、親の発声を模倣する段階」。

## 検証仮説

> 人間の乳児が持つとされる4つの本能のみを与えた場合、あらかじめ答えを教え込まなくとも、太郎は親の発話を模倣する行動（オウム返し）を自ら獲得する。

## 成功基準

親の入力と完全一致する出力を**連続10回**達成。

## 版管理

| 版 | 概要 | 状態 |
|----|------|------|
| **A1** | 最初の試作（DEMO）。コア4本能で動作検証 | 🔧 開発中 |
| A2 | A1の知見をもとに改良（統計学習・比較実験等） | 未着手 |

## コア4本能（A1）

1. 模倣衝動（真似したい）
2. 予測（予測誤差最小化）
3. 報酬予測誤差（ドーパミン）
4. 社会的報酬（親の笑顔）

詳細は [本能リスト](docs/本能リスト.md)、技術仕様は [A1設計メモ](docs/A1_設計メモ.md) を参照。

## 撤退ライン

7月第3週（7/20頃）までにparent_simで平均模倣報酬の明確な上昇傾向が見られなければ方針見直し。
