# D2 設計：逆モデルで「相手の目標」を推論する（触覚のみ・視覚なし）

作成：2026-07-14
位置づけ：[設計_他者理解.md](設計_他者理解.md) のD2段。C3で得た逆モデル(`infer_goal_action`)を、**触覚で知覚した他者**に当てはめ、「相手が何を狙って動いているか」を逆算する。

---

## ゴール
Aが、接触中のBについて **「Bの意図(=狙っている目標)」を触覚だけから当てる**。K_g個の離散目標のうちどれかを、偶然(1/K_g)・persist・floor・shuffle を明確に超えて識別できるか。

## 核心の制約（設計を決める1点）
**Aは相手Bを触覚(56次元＝Aの部位ごとの接触力)でしか感じられない。Bの関節角(qpos)は観測できない。**
- Cの自己モデル`nat_head`は「**自分の**行動→**自分の**固有感覚変化」を予測する。Bの効果を予測する項は持っていない。
- ＝D2の新規部分は「**Bの目標 → Aが感じる触覚**」を橋渡しする成分。これがD2で唯一新しく要る機構。

---

## 実験セットアップ（真値ラベルを作る）
1. **B を目標指向にする**：学習済み自己モデル(`self_A_seed0.pt`)をBにも積む＝両者とも「太郎」。Bに離散目標 g* ∈ {G_1,…,G_{K_g}} を1つ割り当て、`infer_goal_action` で g* に向かって動かす。
   - 目標 = K_g個の代表姿勢(prototype posture)。Bのproprio空間から離れたK_g点をサンプルして固定。**ラベル = どの代表姿勢か**（chance = 1/K_g）。
   - ＝「相手はランダムでなく"意図"を持って動く」を担保（旧d_b2失敗の主因＝相手がランダム、を潰す）。
2. **A は接触を保つ**（後述の接触密度の決定）。Aは触覚列 touch_of_B を時間窓Tで集める。
3. **A が g* を推論**。推論した ĝ が真の g* と一致する率を測る。

## 相手目標を触覚から推論する2つの機構
### (β) 観測モデル＝MOSAIC責任信号【工学ベースライン・測れる】
- Aは較正フェーズで「目標 g_k → 期待される触覚 touch_pred_k」を学ぶ（各目標のBを触って"どう感じるか"を覚える）。
- 推論時：責任信号 r_k = softmax(−‖touch_obs − touch_pred_k‖) を全目標で計算 → **ĝ = argmax r_k**。
- 根拠【人間模倣】MOSAIC(Haruno/Wolpert/Kawato)の責任信号＝観測とモジュール予測の一致度で意図を選ぶ。介護者が「この人のこの動きはこう感じる」を手で覚えるのに対応。
- 長所：確実に測れる・MOSAIC直系。短所：「Cの自己モデルの転用」ではなく**新規に相手用モデルを学ぶ**（転用の主張は弱い）。

### (α) ミラー/共鳴＝自己モデルの転用【人間模倣の本命・難しい】
```
touch_of_B(56) ─[共鳴マップ(新規・小)]→ 「自分の体で言えばこの動き」の推定(proprio空間) ─[Aの逆モデル infer_goal_action]→ ĝ
```
- Bの力を「**もし自分の体がこう動いていたら**」に写し取り(共鳴マップ)、その上で**A自身の逆モデル**で目標を読む。
- 根拠【人間模倣】シミュレーション説/ミラーニューロン＝他者の動きを自分の運動レパートリーに写して意図を理解。介護者が患者の腕を持ち「自分ならこの動作＝Xに手を伸ばしている」と読むのと同型。AとBは同一身体なので対応は自然（ただし力→動きの写像は学習が要る）。
- 長所：**「Cの自己モデルを他者に当てはめる」の本命**（目標読み出しはA自身の逆モデル、新規は共鳴マップだけ）。短所：共鳴マップの学習が要る・希薄接触に弱い。

### 方針（太郎の流儀＝人間模倣ファースト）
- **αを本命として試す。βは"測れる下限/対照"として併走。** αがβに匹敵/上回れば「自己モデル転用」の証拠。αが偶然を超えられなければ、βへ退避し**逸脱(工学近似)としてラベル**して先へ（debugging-workflowに従う）。

## 接触密度の決定（要判断）
現状15%接触では窓内に信号がほぼ無く推論が空回りする。密な接触が要る：
- **案1：sepを詰める/初期姿勢で触れさせる** — 実装最小。ただし"たまたま触れてる"で受動的。
- **案2：Aが能動的に接触を保持（介護者が支える）** — Aを「Bの方へ手を当て続ける」姿勢に固定 or 目標指向で接触維持。**人間模倣・介護に忠実**（hands-on）。実装やや重い。
- 推奨：**まず案1で信号を出せる状態を作り→機構(α/β)の可否を見る→効くなら案2で"介護的な保持"に格上げ**（段階的に人間らしさを上げる）。

## 指標（旧d_b2の対照を流用）
- **主指標**：K-way 目標識別率 vs chance(1/K_g)。
- **対照**：persist(直前触覚の外挿だけで当たらないか)・floor(未学習マップで当たらないか)・shuffle(ラベル混ぜ)。
- C3の教訓：**復元でなく識別/確率で測る**（絶対復元は弱いが区別は頑健）。

## 段階
- **D2a**：接触密度を確保(案1) + β(観測モデル)で K-way 識別が偶然を超えるか＝「触覚から相手意図が読めるか」の可否判定。
- **D2b**：α(ミラー/共鳴＝自己モデル転用)で同課題。αがβに匹敵するか＝「自己モデルを他者に当てはめられるか」の核心検証。
- （→ **D3**：推論した ĝ を `infer_goal_action` に入れて**Bの次の行動を予測** → anticipation 2AFCが偶然/persist/floor/shuffleを超えるか）

## 根拠ラベルまとめ
- 【人間模倣】MOSAIC責任信号／シミュレーション説・ミラーニューロン／介護のhands-on／同一身体の運動対応。
- 【土台=C】`infer_goal_action`(C3逆モデル)をB駆動とA推論の両方に流用。自己モデル`self_A_seed0.pt`を両者に。
- 【逸脱候補】K_g離散目標・代表姿勢の固定・共鳴マップの新規学習・（α不成立時)βへの退避 — 実装時に[人間模倣からの逸脱リスト.md]へ追記。

## 未解決・注意
- 触覚は接触時のみ→推論は接触継続場面に限る（介護的には自然）。
- 共鳴マップ(α)は力→動きの逆問題を含み不良設定になりうる→**識別で測る**ことで回避。
- Bの目標を離散K_g個に絞るのは測定の便宜＝人間の連続的意図からの逸脱（D3以降で連続化を検討）。

## 懸念事項と対照（事前登録 2026-07-14）
D2は「触覚から目標を当てられた」に見えて**別の理由で当たる**罠が多い。本命(α/β)が下の対照を**順に全部超えて**初めて「意図を読めた」と認める。実行前にT・K_g・指標・閾値を固定（p-hack回避）。

| # | 懸念 | 危険 | 潰し方 | 対照(naive) |
|---|---|---|---|---|
| **1** | **相手Bが源泉で読めない**（弱い自己モデルだと`infer_goal_action`が≒ランダム→目標ごとにBの動きが分離しない） | 「A側が読めない」か「Bに意図が無い」か切り分け不能。**低margin時の最大懸念** | **源泉分離性ゲート**：Bの真のqpos軌道(sim真値)が目標で偶然を超え分離するか**先に検証**。不合格ならD2着手不能＝自己モデル強化が先 | 分離しない状態でA側測定＝chance |
| **2** | **静的フィンガープリントの近道**（目標=別姿勢→到達時の接触跡が目標固有→意図推論ゼロで分類可能） | K-way精度は出るが意図理解でない | ①**時間依存精度**（到達前の早い段階で当たるか）②接触部位が重なる目標選定 | **静的分類器**(最終フレーム触覚→目標) |
| **3** | **βの丸暗記**（K_g=3を訓練試行ごと暗記） | 未見試行で崩れる／β>αを誤解釈 | **訓練/テスト分割**（別初期状態・別シードの未見で評価） | 同一試行で訓練=評価の見かけ高精度 |
| **4** | **αの"転用"が空主張**（共鳴マップが大きくβと同義に） | 核心主張が崩れる | Aの逆モデル**凍結**＋共鳴マップ**小容量**＋**アブレ**（逆モデルをランダム初期化に差替え劣化するか） | ランダム初期化逆モデル版α |
| **5** | **接触断続が手掛かり化**（目標で接触率が違う／無接触ゼロを信号扱い） | 接触率で分類＝意図でない | 目標間で接触率を揃える／無接触フレームは**マスク**／接触率を報告 | 接触率のみ分類器 |
| **6** | **同一身体の下駄**（A=B完全同一で転用が自明有利） | αを過大評価 | D2は同一でOK(最易ケース)だが**限界明記**。将来B身体を微摂動 | （将来の摂動B） |
| **7** | **p-hacking**（当たるまでT/K_g/閾値を試す） | 偶然を成功と誤認 | **事前登録**（T・K_g・指標・閾値を実行前固定、shuffleを帰無分布） | 事後選択版 |

### 成功判定（階段状）
```
偶然(1/3) < shuffle < 静的フィンガープリント < 接触率のみ   ← ここまで naive/交絡の壁
                                              ↓ これを超えて初めて
                                    α/β(本命) = 真の意図読み
  追加検証: 早い段階でも当たる(時間依存) / 未見試行で当たる(汎化) / 逆モデル凍結が効く(転用)
```
naive版(静的分類器・接触率分類器・shuffle)を明示的に実装し、本命がそれらを上回った**差分だけ**を「意図理解の正味」として報告。
