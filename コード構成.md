# コード構成

各スクリプト・モジュールの役割をまとめたものです。太郎は「人間の脳・身体を模した部品」で
構成されており、ファイル名も可能な限り対応する人間の部位に合わせています。

目標A（オウム返し）は `A/`、目標B（初語）は `B/` 以下にあります。以下は現在進行中の
**目標B（`B/`）**を中心に説明します。

---

## 実行スクリプト（`B/` 直下）

シミュレーションを回したり、検証したりする入口。すべて `python <ファイル名> [引数]` で実行します。

| ファイル | 役割 |
|---|---|
| `run_trace.py [日数]` | **まず試すならこれ。** 短期間（既定7日）のシミュを回し、リプレイビューア用のログを出力する |
| `run_12months.py [試行ID]` | 12ヶ月（1年）分のシミュレーション |
| `run_24months.py` | 24ヶ月（2年）分のシミュレーション |
| `run_b6_4_730.py <seed> [gain]` | B6-4（語↔内的状態の連合）の2年シミュ。gain=0で連合オフ＝除去テスト |
| `run_b5_730.py` / `run_b5_5_730.py` / `run_b6_730.py` / `run_taste730.py` | 各段階（B5結晶化・B6分節・味覚）の2年シミュ |
| `run_comprehension_probe.py` | 「語→対象の予期（理解）」を測る理解テスト |
| `run_b212_verify.py` / `run_b213_seed.py` | 特定版の複数シード検証 |
| `run_diagnostic.py` | hungerを固定して喃語内容への影響を直接測る診断実験 |
| `calibrate_hunger.py` / `verify_meals.py` | 空腹速度の較正・授乳回数の確認 |
| `perf_setup.py` | BLASスレッド数制限・プロセス優先度の調整（各runから読み込む） |
| `build_trace_index.py` | リプレイ用トレースの索引作成 |

## `B/src/environment/` — 環境（親）と太郎をつなぐループ

| ファイル | 役割 |
|---|---|
| `core_b.py` | **目標Bの中核。** 太郎の1ステップ（聞く→学習→発声→報酬）と身体シミュレーションを回す |
| `parent_sim_b.py` | 親シミュレータ＋シミュレーション全体の駆動（`run_simulation_b`） |
| `cli_b.py` | 人間が親役になって直接話しかける手動モード |
| `core.py` / `parent_sim.py` / `cli.py` | 目標A（オウム返し）用の旧環境 |
| `ablation.py` | 部品をON/OFFして寄与を測る除去テスト用 |

---

## `B/src/taro/brain/` — 脳

| ファイル | 対応する人間の部位・役割 |
|---|---|
| `cortex.py` | 大脳皮質。予測する単一の再帰ネット（GRU）。聞いた音を処理し、口の動きを出力 |
| `basal_ganglia.py` | 大脳基底核。方策勾配（REINFORCE）＋予測誤差最小化で重みを更新 |
| `cerebellum.py` | 小脳。口の動き↔音の対応を体験から学び、反復で自動化（結晶化）する |
| `left_frontal_lobe/brocas_area.py` | ブローカ野。発話計画（GODIVAモデル）を立てて実行する |
| `hippocampus.py` | 海馬。経験を蓄え、睡眠中に大脳皮質へ定着させる（記憶リプレイ） |
| `insula.py` | 島皮質。身体の内受容感覚を脳へ伝える経路 |
| `lexicon.py` | 聞いた音を統計的に分節して語形を蓄える辞書。語↔内的状態の連合（B6-4）もここ |

### `B/src/taro/brain/instincts/` — 本能（学習を駆動する生得の仕組み）

| ファイル | 役割 |
|---|---|
| `dopamine.py` | ドーパミン。報酬予測誤差（強化学習のTD誤差） |
| `prediction.py` | 予測処理。次に来る音を当てられると心地よい（Friston） |
| `imitation.py` | 模倣衝動。親の発声にどれだけ似ているか |
| `homeostasis.py` | 恒常性。つらさ（空腹など）が解消されると報酬 |
| `hedonic.py` | 快。甘み（味覚）を感じると生得的に心地よい（liking） |
| `locus_coeruleus.py` | 青斑核。ノルアドレナリンで探索↔活用のバランスを調整 |
| `habituation.py` | 馴化。同じ発声の繰り返しに飽きる |
| `critic.py` | 状態依存の価値予測（Actor-Criticのcritic） |
| `reward.py` | 各本能の報酬を1つに合成する |

---

## `B/src/taro/body/` — 身体

| ファイル | 役割 |
|---|---|
| `stomach.py` | 胃。食べ物を消化し、空腹度を生む |
| `blood_vessel.py` | 血管。血中栄養素を蓄え、低いと空腹感になる |
| `lungs.py` | 肺。空気をため、発声できる長さを決める |
| `internal_state.py` | 内受容感覚。臓器の信号を「つらさ・空腹・眠気・不快」に統合 |
| `adenosine.py` | アデノシン。覚醒時間に応じて溜まり、眠さを生む |
| `tongue.py` | 舌。甘み（味覚）を検知する |
| `vocal_tract.py` | 声道。口の4パラメータ（調音位置・様式・声帯・母音）を音に変換。発達段階で解禁される音が増える |
| `stamina.py` | 体力（目標A系で使用） |

## `B/src/` その他

| ファイル | 役割 |
|---|---|
| `logger.py` | 毎ターンのデータ・喃語・イベントをJSONLに記録し、学習曲線を出力 |
| `sim_clock.py` | シミュレーション内の時間・発達年齢の管理 |
| `archive.py` | 脳のスナップショット保存・復元（成長アーカイブ） |

## 設定ファイル（`B/config/`）

- `config.yaml` … 脳のハイパーパラメータ・報酬の重み・身体の生理定数など
- `schedule.yaml` … 親の授乳スケジュールなど環境側の設定
