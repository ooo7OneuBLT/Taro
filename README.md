# プロジェクト「太郎」

人間の発達段階を1段ずつ模倣しながら育てるAI。

## 哲学

* 太郎の内部には**人間の乳児が生まれつき持つ本能だけ**を実装する
* 人間に存在しない便利機能（`print(input())`等）は使わない
* すべての機能に根拠ラベルを付ける：**【人間模倣】** or **【既存AI研究】**（逸脱は⚠️明記）
* 学習の工夫は太郎の内部ではなく「環境（親の話し方）」側に置く

## 前提

* 太郎は**日本語が母語の日本人・男**の設定
* 長期ビジョン：人間と同じように育てたAIは人間社会になじめるのではないか

## 発達段階

|段階|目標|状態|
|-|-|-|
|**A**|オウム返し（親の発言を模倣する）|🔧 A1 試作中|
|B 以降|人間の発達順に沿って拡張（要調査）|未着手|

詳細は [発達段階ロードマップ](A/docs/発達段階ロードマップ.md) を参照。

## ディレクトリ構成

```
Taro/
  README.md          ← このファイル
  A/                 ← 目標A（オウム返し）の全作業
    config/          ← ハイパラ・本能定義
    src/             ← ソースコード
    archive/         ← 脳スナップショット（成長アーカイブ）
    logs/            ← 自動ログ・学習曲線
    docs/            ← 計画書・設計メモ・研究日誌
```
