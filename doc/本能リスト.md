# 太郎の本能リスト（プロジェクト全体）

太郎に与えた「生得の本能」と「生まれつき持つ身体・脳の部品」の一覧。発達（目標A→B→…）を通じて累積する。各項目に人間での根拠・実装場所・導入した目標・有効段階を記録する。

太郎の中身は**本能のみ**（答え・ラベルは注入しない）という設計哲学に基づく。ラベル凡例：【人間模倣】＝人間の仕組みに対応、【既存AI研究】＝工学的手法。

## 本能（報酬・学習を駆動する生得の動機）

### 1. 模倣衝動（真似したい）— 導入: 目標A
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 新生児模倣（Meltzoff & Moore, 1977）。近年反論あり（Oostenbroek et al., 2016）＝未決着のため仮説的本能として採用 |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/imitation.py |
| 実装 | 親の発話と太郎の出力の類似度 → 内的報酬 r_imit ∈ [0,1] |
| 有効段階 | A〜（他の報酬源が育つと相対的に小さくなる＝創発的減衰） |
| ラベル | 【人間模倣】 |

### 2. 予測（予測誤差最小化）— 導入: 目標A
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 予測処理／自由エネルギー原理（Friston）。脳は次の入力を絶えず予測する |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/prediction.py |
| 実装 | 次トークンの予測が当たるほど報酬 r_pred |
| 有効段階 | A〜（恒久） |
| ラベル | 【人間模倣】 |

### 3. ドーパミン（報酬予測誤差）— 導入: 目標A
| 項目 | 内容 |
|------|------|
| 人間での根拠 | ドーパミンニューロンが「もらえた報酬 − 予想した報酬」を発火（Schultz, 1997）。RLのTD誤差と数式一致 |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/dopamine.py |
| 実装 | 学習信号 δ = 実報酬R − baseline。全学習を駆動 |
| 有効段階 | A〜（恒久） |
| ラベル | 【人間模倣】＝【既存AI研究】 |

### 4. 社会的報酬（親の反応で安心する）— 導入: 目標A
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 新生児は笑顔・親の声・対乳児発話・視線・随伴性に生まれつき反応する |
| 場所 | 環境（親の入力） |
| ファイル | parent_sim.py／parent_sim_b.py |
| 実装 | 親の反応（笑顔の強さ等）を連続値 r_social ∈ [0,1] で入力 |
| 有効段階 | A〜（恒久） |
| ラベル | 【人間模倣】 |

### 5. 馴化（飽き）— 導入: 目標B
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 胎児・新生児にもある馴化（Sicard-Cras, 2022） |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/habituation.py |
| 実装 | 同じ出力を繰り返すとマイナス報酬（飽きのペナルティ） |
| ラベル | 【人間模倣】 |

### 6. 青斑核（探索レベルの調整）— 導入: 目標B
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 青斑核のノルエピネフリン放出で探索モード制御（Aston-Jones & Cohen 適応ゲイン理論）。B2-8で成熟による結晶化（Gopnik 探索→活用の移行）を追加 |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/locus_coeruleus.py |
| 実装 | 最近の報酬が低→探索増、高→活用。成熟で探索天井 1.0→0.3 に低下（mature） |
| ラベル | 【人間模倣】（成熟カーブの決め打ちは⚠️逸脱） |

### 7. 恒常性（体の状態を元に戻したい）— 導入: 目標B
| 項目 | 内容 |
|------|------|
| 人間での根拠 | ホメオスタシス（Cannon, 1932）。生物の基本機能・生得的 |
| 場所 | 太郎の中身（本能） |
| ファイル | instincts/homeostasis.py |
| 実装 | 体のつらさが下がった「ほっとした分」が報酬。「まんま」という音に意味が生まれる土台 |
| ラベル | 【人間模倣】 |

### 8. 自発喃語（脳の現在の分布からの声出し）— 導入: 目標B（B-3）
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 乳児は0〜6か月、穏やかな時間に発話計画なしで声を出し、聴覚フィードバックで声道の使い方を強化する |
| 場所 | 環境（parent_sim_b.py）＋太郎の脳（core_b.py の self_babble） |
| ファイル | environment/core_b.py |
| 実装 | 起きて泣いていないとき脳の確率分布から音をサンプリング、自分の声を聞いて再学習（自己強化）。1日約240回 |
| ラベル | 【人間模倣】 |

### 9. 学習進度（progress）＝好奇心 — 導入: 目標C
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 内発的動機づけの「学習進度(learning progress)」仮説（Oudeyer & Kaplan）。乳児は"難しすぎず簡単すぎない"＝**予測誤差が減りつつある**対象に注意を向ける |
| 場所 | 太郎の中身（本能） |
| ファイル | `C/scripts/run_c_metrics_ac_lr.py`（⚠️**モジュール化されていない**。要抽出） |
| 実装 | `rew = pe_slow − pe_fast`（遅い移動平均−速い移動平均）＝誤差が減っていれば正の報酬 |
| 有効段階 | C〜（**目標Eのリーチ創発でも動機の本体**。§目標E設計 §10.1） |
| ラベル | 【人間模倣】＝【既存AI研究】 |

### 10. Goal Babbling（目標を思い描いて動く）— 導入: 目標C
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 乳児の運動学習は無作為ではなく「行きたい先」を持った探索（goal babbling, Rolf & Steil）。到達運動の発達研究に対応 |
| 場所 | 太郎の中身（本能） |
| ファイル | `C/scripts/run_c_metrics_ac_lr.py:266` `infer_goal_action()`（⚠️モジュール化されていない） |
| 実装 | 目標状態gを置き、順モデルを通して「gに近づく行動」を推論時の勾配降下で求める |
| 有効段階 | C〜 |
| ラベル | 【人間模倣】 |
| ⚠️逸脱 | 探索/目標指向の切替が固定頻度だった（[逸脱リスト 2026-07-14](人間模倣からの逸脱リスト.md)）。創発版に置換済みだが配分式は依然根拠なし |

### 11. 努力コスト（動くのは疲れる）— 導入: 目標C（taro-C5）
| 項目 | 内容 |
|------|------|
| 人間での根拠 | 運動の代謝コスト最小化。生物は同じ結果なら少ない筋活動を選ぶ |
| 場所 | 太郎の中身（本能） |
| ファイル | `C/scripts/run_c_metrics_ac_lr.py:638`（環境変数 `C_EFFORT` で強度指定） |
| 実装 | `rew = rew_task − λ×Σ(行動²×筋のgear重み)`＝強い筋ほどコストが高い |
| 有効段階 | C5〜（既定λ=0＝**通常はOFF**。運動品質の調査で導入し、progress報酬が本命と判明） |
| ラベル | 【人間模倣】 |

## 原始反射（生得の反射弓。皮質＝方策を経由しない）— 導入: 目標E

| 反射 | 状態 | ファイル | 根拠 |
|------|------|---------|------|
| **VOR（前庭動眼反射）** | ✅ 実装済み | `E/scripts/e_vor.py` | 三ニューロン弓＝脳幹で完結。利得1.03（1〜4ヶ月児の実測、Finocchio et al. 1991）。[引用.md](../引用.md) |
| **把握反射（palmar grasp）** | ⚠️ **設計のみ・未実装** | — | 確立した原始反射リストから選択（StatPearls）。接触を持続させ随伴性を経験させる。[全体設計 §10.6](../E/docs/全体設計_目標E.md) |
| ATNR（非対称性緊張性頸反射） | ❌ **採用しない（根拠あり）** | — | 健康新生児108名中**真の反射は8%のみ**（Vassella & Karlsson 1962）／健康児で**優位な姿勢になったことがない**（Ferrari 2007）／最新レビューは**言及すらしない**（Hadders-Algra 2018）。[引用.md](../引用.md)・[参考文献リスト §目標E-14](参考文献リスト.md) |
| （代替）視覚で自分の腕を保持する傾向 | ❌ 未実装・**検討に値する** | — | 新生児は**腕が見えるときだけ**外力に抗して腕を上げ続ける（van der Meer et al. 1995, *Science*）＝ATNRより実験的に強い。太郎の自己モデル路線とも整合 |
| モロー反射・ルーティング・吸啜 | ❌ 未検討 | — | 同上 |

⚠️ **VORの実装上の注意**：方策の出力次元は減らしていない（眼球ぶんを受け取るが捨てる）。C5の学習済みモデル（n_actuators=90）を読めなくしないため。実効利得は0.64で文献値1.03に未達＝原因は頭が動きすぎることによるトルク飽和（[逸脱リスト 2026-07-20](人間模倣からの逸脱リスト.md)）。

## 生得的な身体・脳の部品（本能ではないが生まれつき持つ仕組み）

| 部品 | ファイル | 導入 | 何をしているか |
|------|---------|------|-------------|
| 胃 | body/stomach.py | B | 食べ物を受け取り消化で空になる容器。空腹度を出す |
| 肺 | body/lungs.py | B | 空気をため発声で使う。発声できる長さを決める |
| 内受容感覚 | body/internal_state.py | B | 臓器の信号を「つらさ」に統合して脳へ渡す |
| 島皮質 | brain/insula.py | B | 体の感覚を大脳皮質へ伝える経路 |
| 泣き | brain/cortex.py | B | つらさに比例して声を出す。最初は反射、やがて意図的 |
| 血管 | body/blood_vessel.py | B-3 | 胃の吸収量を血中栄養素として蓄える。低いと空腹感 |
| アデノシン（眠さ） | body/adenosine.py | B-4 | 覚醒時間に応じて溜まり睡眠で減る。眠さをメカニズムから算出 |
| 海馬（睡眠リプレイ・記憶定着） | brain/hippocampus.py | B-5 | 経験を覚醒中に蓄積し睡眠移行時に大脳皮質へ定着（補完学習系CLS, McClelland 1995）。B3以降は満腹予期のTD学習もここで |
| 満腹予期ヘッド | cortex.py (satiety_head) | B2-10 | 聞いた語＋体の状態から「もうすぐ満腹になるか」を予測。B3で教師ラベルを廃しTD学習に |
| 舌（味覚センサー） | body/tongue.py | B4 | 甘み（栄養）を検知。快は出さない |
| 快（甘み→liking） | brain/instincts/hedonic.py | B4 | 甘みに生得的に快を感じる。ドーパミン(wanting)と別信号（Berridge の liking/wanting 区別） |
| 小脳（運動自動化） | brain/cerebellum.py | B5-3 | 反復練習回数から自動化度を計算し、進むほど運動ノイズを縮小（報酬なしの結晶化, Vihman VMS） |
| 辞書（統計的分節・状態連合） | brain/lexicon.py | B6-1／B6-4 | 予測しやすさの谷で単語を教師なし分節（Saffran 1996）＋語↔内的状態の連合（cross-situational learning, Smith & Yu 2008） |
| Critic（状態依存の価値予測） | brain/instincts/critic.py | B-11 | body_state から価値V(s)を予測しドーパミンの baseline を置換（Actor-Critic化） |
| 声道（口） | brain/vocal_tract.py | B | 調音点・調音法・声帯振動・母音の4パラメータで音を作る。「ま」と「ぱ」が1パラメータ差＝身体的な近さが脳に生まれる |
| 大脳基底核 | brain/basal_ganglia.py | A2-10 | ドーパミンで行動を強化・抑制する行動選択。⚠️重み更新はbackprop【逸脱】 |

## 発達で消える／減衰する本能

人間の乳児には把握反射・モロー反射など生後4〜6か月で消失する原始反射がある。太郎に直接対応するものは少ないが、「足場として働き役目を終えると弱まる」原理は、模倣衝動の相対的減衰として自然に再現される。

   `C/scripts/run_c_metrics_ac_lr.py` に直接埋め込まれており、`taro_core` の部品になっていない。
   太郎の**動機の中核**がスクリプトの中にあるのは構造として不健全。要抽出。
