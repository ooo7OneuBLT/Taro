# 注意すべき機能リスト（プロジェクト全体）

**逸脱ではない**（＝人間模倣に忠実で採用してよい機能）。ただし**常時稼働・現状は挙動をほとんど変えない・作りが粗い**などの理由で、**壁に当たったときの原因候補として見落としやすい**機能を台帳化する。

- 「人間模倣でないと分かっているが割り切っている所」は別台帳 → `人間模倣からの逸脱リスト.md`
- 「壁の原因が過去の機能のことがある」→ `C/docs/検証の落とし穴チェックリスト.md` 項目8 と対で使う。**デバッグの候補列挙で、この表の機能を必ず1度は疑う。**

判定の目安：✅採用（人間模倣に忠実）だが、⚠️「既定ONで黙って効く」「効果が現状ほぼ不変」「作りが粗い」のいずれかを持つ → ここに載せる。

## 目標C（自己内部モデル・身体／MIMo）

### C-注1. 運動小脳（MotorCerebellum, 連続版）＝運動の自動化・結晶化

| 項目 | 内容 |
|---|---|
| **場所** | `C/src/brain/cerebellum_motor.py`、配線＝`C/scripts/run_c_metrics_ac_lr.py`（`C_CEREBELLUM`、**既定ON**） |
| **何をする** | 方針(`motor_head`)の行動を状態(z)から真似るフィードフォワード地図。馴染んだ状態ほど自動化重み w↑で滑らかな出力に置換＋探索ノイズ減＝結晶化（VMS, Vihman／小脳のフィードフォワード制御, Miall & Wolpert 1996）。Bの離散小脳の連続版 |
| **なぜ採用（逸脱でない）** | 運動の自動化は**人間模倣に忠実な機能**。NE（全体ノイズ）とも順モデル（予測）とも別機構になるよう設計済み |
| **⚠️注意1：良性だが現指標では不変** | ON/OFFアブレーション（2シード・1時間）で**発散・運動フリーズ・agency崩壊なし＝安全**を確認。一方 margin/corr/persist/agency は **ON≈OFF**（Cでは結晶化系が元々ほぼ無効＝報酬が先にノイズを下限へ落とすため）。＝**入れても壊れないが、今は効果を出さない** |
| **⚠️注意2：常時稼働（黙って効く）** | 既定ONで毎ステップ約26%ブレンド（w≈w_max·e⁻¹）。**壁に当たったとき「小脳が黙って効いている」可能性を見落としやすい**。デバッグの候補列挙で必ずON/OFF比較を入れる |
| **⚠️注意3：作りが粗い** | 自動化重み w が**全身で1つ**（`gate`が90関節の平均誤差から1個のwを出す）。本物の小脳の**体部位ごとの地図（somatotopy）を持たない**＝「腕はよく練習した／脚はまだ」の差をつけない |
| **本当の出番（いつ効くか）** | ①**行動→感覚の結びつきを強めた後**（＝逆モデルB道の前提。action再帰化は済、予測幅短縮など）②**部位ごとのw（somatotopy）に精緻化**した後。この2つの土台ができてから再評価する |
| **検証記録** | `C/docs/研究日誌.md`（2026-07-14 小脳アブレーション）／数値は `logs/C/cerebellum_ablation/` |

### C-注2. 睡眠リプレイ（記憶定着）＝自己モデル確立の本命機構・既定ON化

| 項目 | 内容 |
|---|---|
| **場所** | `C/src/brain/`（`MotorHippocampus`＋`TaroBrainWithMotor.consolidate`）、配線＝`run_c_metrics_ac_lr.py`（`C_REPLAY`。**2026-07-15に既定OFF→ON**、600step毎に200バッチ再生） |
| **何をする** | 覚醒中の経験を貯め、周期的に再生して順モデルを固める（CLS McClelland 1995／目標Bの海馬と同原理）。＝**margin+11→+48・corr0.44→0.56・agency83→95%の頭打ち突破の主因** |
| **⚠️注意（重大）：長らく既定OFFだった** | 本命機構なのに`C_REPLAY`既定が`"0"`で、素の実行は**リプレイ無し(margin~11)**になっていた。＝「+48」は`C_REPLAY=1`のときだけ。これが原因で「+48は普通のベースライン」と誤解し、Dの自己モデル(+11)を"壊れている"と長時間追った。**既定ONに修正済み**（`C_REPLAY=0`はアブレーション用） |
| **⚠️注意：大量学習への寄り（未解決）** | リプレイ:覚醒≒43:1で復習過剰の懸念（質問リストE章）。直近/顕著さ重み無し・一様サンプルの是非も要検討 |
| **検証記録** | `C/docs/研究日誌.md`（2026-07-15 順/逆の天井点検）／`C/docs/C/検証指標の一覧と変遷.md` §5-6 |
