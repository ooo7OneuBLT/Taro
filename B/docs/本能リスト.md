# 太郎の本能リスト（目標B）

太郎に与えた本能の一覧。モデルAから引き継いだものと、Bで追加したものを記録する。

---

## Aから引き継いだ本能

### 1. 模倣衝動（真似したい）

- 人間での根拠：新生児模倣（Meltzoff & Moore, 1977）。学術的に未決着だが仮説的に採用
- 場所：太郎の中身（本能）
- ファイル：instincts/imitation.py
- 何をしているか：親の発話と太郎の出力がどれくらい似ているかを計算して報酬にする

### 2. 予測（次に何が来るかを当てたい）

- 人間での根拠：予測処理 / 自由エネルギー原理（Friston）
- 場所：太郎の中身（本能）
- ファイル：instincts/prediction.py
- 何をしているか：次の文字の予測が当たるほど報酬が出る

### 3. ドーパミン（報酬予測誤差）

- 人間での根拠：ドーパミンニューロンの発火＝報酬予測誤差（Schultz, 1997）
- 場所：太郎の中身（本能）
- ファイル：instincts/dopamine.py
- 何をしているか：「もらえた報酬 − 予想していた報酬」を計算する。これが全学習を動かす信号

### 4. 社会的報酬（親の反応で安心する）

- 人間での根拠：新生児は笑顔・親の声・視線に生まれつき反応する
- 場所：環境（親の入力）
- ファイル：parent_sim.py / parent_sim_b.py
- 何をしているか：親が世話をしてくれたとき、その反応を報酬として受け取る

### 5. 馴化（飽き）

- 人間での根拠：胎児・新生児にもある馴化メカニズム（Sicard-Cras, 2022）
- 場所：太郎の中身（本能）
- ファイル：instincts/habituation.py
- 何をしているか：同じ出力を繰り返すとマイナスの報酬（飽きのペナルティ）

### 6. 青斑核（探索レベルの調整）

- 人間での根拠：青斑核がノルエピネフリンを放出して脳の探索モードを制御する
- 場所：太郎の中身（本能）
- ファイル：instincts/locus_coeruleus.py
- 何をしているか：最近うまくいっていない→もっと色々試す。うまくいっている→今のやり方を続ける

## Bで追加した本能

### 7. 恒常性（体の状態を元に戻したい）

- 人間での根拠：ホメオスタシス（Cannon, 1932）。生物の基本機能。生得的
- 場所：太郎の中身（本能）
- ファイル：instincts/homeostasis.py
- 何をしているか：体のつらさが下がったとき（例：お腹が満たされた）に報酬が出る。「ほっとした分」が報酬
- なぜ必要か：これが「なぜ『まんま』という音に意味が生まれるか」の答え。「まんま」の後に空腹が解消される体験を繰り返すと、「まんま」が大事な音になる

### 8. 自発喃語（脳の現在の分布からの声出し）

- 人間での根拠：乳児は0〜6か月に穏やかな時間に発話計画なしで声を出す。聴覚フィードバックで声道の使い方を強化する
- 場所：環境（parent_sim_b.py）＋ 太郎の脳（core_b.py の self_babble()）
- ファイル：environment/core_b.py（self_babble メソッド）
- 何をしているか：起きていて泣いていないとき、脳の今の確率分布から音をサンプリング。自分の声を聞いて再学習する（自己強化）
- B-3で実装済み。1日約240回（人間の新生児に近い）
